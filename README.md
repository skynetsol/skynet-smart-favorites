# SkyNet Smart Favorites

SkyNet Smart Favorites is a WooCommerce wishlist plugin that helps customers save, organize, and manage favorite products for later. It combines fast guest-session support, flexible list handling, and conversion-friendly wishlist actions for product pages and shop loops.

## Description

SkyNet Smart Favorites adds a lightweight wishlist experience to WooCommerce stores. Customers can:

- Save favorite products from shop and product pages.
- Create and manage multiple wishlist lists.
- Keep wishlist items during guest browsing sessions.
- Move wishlist items directly to the cart.

The plugin is built for a seamless WooCommerce workflow and includes admin settings for display, notification, and page controls.

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin from the WordPress Plugins screen.
3. Go to the **SkyNet Smart Favorites** admin menu.
4. Configure settings and verify the wishlist page is created.

## Frequently Asked Questions

### Can I use the wishlist without WooCommerce?

No. SkyNet Smart Favorites requires WooCommerce to be installed and active.

### Does it support guest users?

Yes. Guest wishlist data is stored in the WooCommerce session while visitors browse.

### Will wishlist items sync after login?

Yes. The plugin includes login-sync foundations to preserve wishlist data after users log in.

### How do I display the wishlist page?

Use the shortcode: `[ssf_wishlist]`. You can also use `[ssf_wishlist_icon product_id="123"]` to show an add/remove wishlist icon.

### Where are the settings located?

Open the **SkyNet Smart Favorites** menu in the WordPress admin to configure global behavior, display options, and page settings.
